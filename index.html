PyOfWave is an implementation of Google-designed Wave communication platform (a merge of eMail, wiki, IM, and forums) built upon Twisted Python.

It will come with a jQuery/Agility.js web client to add to your site. This client aims to be User Freindly by following the basic guidelines of Apple and GNOME and by avoiding wasting visual space.

It is the active branch of the PyGoWave client, but is not functioning fully yet.

Dependencies

  • Python 2.7+
  • Twisted
  • Python XMPP Server
