Wave is an all in one Internet communication service. It is a merge of eMail, IM, forums, and wikis resulting in inboxes of decentralized live, fully editable, forum like structured messages. It was designed by then Google Australia employees Lars and Jens Rasmussen and announced at Google I/O in 2009.

PyOfWave is an implementation of Wave using the Python language, commited to a new future for this great and flexible technology.

It will come with a highly brandable/themable User Freindly web client to add to your site. Like Apple Inc. and GNOME, we use User Centered Design and aim for high consistancy with the outside world, the OS, and itself.

It is the active branch of PyGoWave, but is not functioning fully yet.

Dependencies

  • Python 2.7+
  • Twisted
  • Python XMPP Server
