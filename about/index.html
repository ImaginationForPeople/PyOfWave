About

PyOfWave (POW) aims to bring the Google designed communications platform back in a healthy, grass roots, decentralized fashion. We aim to be extendable, decentralized, sufficintly fast, flexible, and user freindly.

We make full use of XMPP not only for Federation (as Google did) but for clients both native and offline. XMPP already provides us a well tested live decentralized messaging system with web bindings, extensibility, and security forming a great base for Wave.

For the geeks and corperations using POW to offer Wave services we offer flexibility. POW will fit whatever persistance, authentication, and protocol setup you have. Furthermore using our settings system, you don't even need to touch POW for all of this.

For end user's, we will offer a web interface inspired by the advice published by Edward Tufte, Apple Inc., and GNOME to offer a superior user experience. The dominant charactoristics we aim for in the web interface are:

  • User Centered Design
  • Consistancy - with world, OS (through semantics), and itself
  • Minimal UI, maximum functionality
  • Information dense - every visual element is both aesthetic and communicative
